# UI/UX 改进计划

基于 UI/UX Pro Max 设计原则，对拼豆编辑器进行系统性的 UI 改进。

---

## 设计原则

参考 [UI/UX Pro Max](https://skills.sh/kimny1143/claude-code-template/ui-ux-pro-max) 的核心原则：

> **「全部ちゃんとやる」がAI生成感の正体。「何を捨てるか」がデザイン。**

### 5 个铁则

1. **作る前に疑え** - 全要素に「本当に要る？」を問え
2. **主役を1つ決めろ** - 各セクションで「一番見せたいもの」を1つだけ
3. **70点を並べるな** - 1つを120点、残りを60点にしろ
4. **「できます」より「やめましょう」** - 迷ったら削れ
5. **批判してから作れ** - 現状の問題点を3つ以上挙げてから改善案を

## 设计检查清单

### 视觉层次
- [ ] 每个区域有明确的视觉主角
- [ ] 次要元素不抢主角色彩
- [ ] 删除任何"可有可无"的装饰

### 间距系统
- [ ] 使用统一的间距单位
- [ ] 分组元素之间有足够的呼吸空间
- [ ] 避免均匀分布，让重要的更突出

### 颜色使用
- [ ] 限制主色调使用数量
- [ ] 同色系搭配确保足够对比度
- [ ] 避免渐变和光效作为默认选择

### 性能
- [ ] 避免不必要的动画
- [ ] 使用 CSS 过渡而非 JS 动画
- [ ] 移除 hover:scale 等重效果
