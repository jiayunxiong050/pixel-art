# MARD Pixel Art Editor - 开发进度

> 最后更新：2026-03-31

---

## 已完成 ✅

### P0 - 核心功能
- [x] **图片导入 + 自动像素化** — 上传图片，按精度参数生成像素网格，自动居中放置
- [x] **画布渲染** — 棋盘格背景（大格 10×10 区分），CSS 实现（不再用 2500 个 Fabric.Rect，性能已优化）
- [x] **拖拽移动图层** — Fabric.js 选中拖拽，`object:modified` 保存格坐标 `offsetX/Y`
- [x] **画布尺寸调整** — 可调 5~300 格，实时生效
- [x] **精度调整** — 可调 5~300，修改后自动重新像素化（保留原始图片，支持多次重像素化）

### P0 - 导出
- [x] **预览弹窗** — 导出前先预览效果
- [x] **单图层居中导出** — 单张图时内容自动居中于画布，导出 PNG
- [x] **品牌色号标注** — 每个格子显示对应品牌的色号（而非序号）
- [x] **购物清单** — 底部图例列出所有颜色及其用量

### P1 - 辅助功能
- [x] **品牌切换** — MARD / COCO / 漫漫 / 盼盼 / 咪小窝
- [x] **显示/隐藏编号** — 画布格子内色号开/关
- [x] **AI 去背** — 基于 @imgly/background-removal，识别主体并去除背景
- [x] **图层面板** — 右侧显示图层列表，支持选择、切换 XY 坐标
- [x] **删除图层 / 清空画布**
- [x] **加载状态提示** — 上传、像素化、去背各阶段显示 loading 提示
